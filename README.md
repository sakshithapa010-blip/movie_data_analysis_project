# 🎬 Movie Industry Data Analysis
Analyzed a dataset of 3,000 movies to provide strategic recommendations for a rookie film producer.

## 🚀 Key Features
- **ROI Analysis:** Identified the most cost-effective languages and genres.
- **Data Cleaning:** Handled missing values and parsed nested JSON strings in 'crew' and 'cast' columns.
- **Top Talent Tracking:** Identified bankable actors and successful Director-Actor pairings.

## 🛠️ Tech Stack
- **Python** (Pandas, NumPy)
- **Visualization:** Matplotlib, Seaborn
- **Environment:** Google Colab / Jupyter Notebook

## 📈 Key Insight
The analysis revealed that while Hollywood has the highest gross revenue, regional languages often yield a higher **Return on Investment (ROI)** for small-scale productions.
